# Media Library
	A Library for adding, deleting and updating the media files  like movies, music, games etc. into database

## Requirements:
	  python >=3.6
  	  pycharmIDE
  	  Ananonda 
  	  pyQt5
  
## Run Command
	 python mediaLibrary.py
